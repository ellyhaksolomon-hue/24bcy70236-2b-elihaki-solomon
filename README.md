## Aim
To create a dynamic product filter that displays products based on the selected category using a dropdown menu.

## Objectives
- Implement filterable product data using JavaScript
- Create a sortable dropdown UI for product categories
- Dynamically update the product list based on selection
- Design responsive product cards using CSS Grid
- Apply smooth UI transitions for better user experience

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
- Vercel (Deployment)
- GitHub (Version Control)

## About the Program
This project demonstrates the use of JavaScript DOM manipulation, event handling, and CSS Grid layout to build a dynamic product filtering system. Users can filter products by category using a dropdown menu, and the displayed product cards update in real time without page reload.

## Folder Structure
index.html
style.css
script.js
README.md


## How to Run the Program
1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Use the dropdown menu to filter products

## Output
The application displays a list of products with name, price, and category. Selecting a category filters the displayed products dynamically.

## Deployment
The project is deployed on Vercel using GitHub integration.
